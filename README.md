# AmoPoint Test

Тестовое задание на вакансию PHP-разработчика.

## Задание 1

Необходимо реализовать Laravel-проект, в котором:

1. Консольная команда каждые 5 минут получает данные из внешнего API и сохраняет их в таблицу БД.
2. Route отдает массив записей таблицы в формате JSON.

API для реализации:

`https://official-joke-api.appspot.com/random_joke`

## Что реализовано

В проекте реализован сбор случайных шуток из внешнего API с сохранением в таблицу `jokes`.

Сохраняются поля:

- `id`
- `external_id`
- `type`
- `setup`
- `punchline`
- `created_at`
- `updated_at`

Логика работы:

- команда `php artisan jokes:fetch-random` получает одну случайную шутку из API и сохраняет её в БД
- Laravel Scheduler запускает эту команду каждые 5 минут
- каждая шутка сохраняется отдельной строкой без проверки на дубли
- `GET /api/jokes` возвращает JSON-массив из 100 записей
- `GET /jokes` показывает сохранённые шутки в HTML-таблице с пагинацией и сортировкой

## Routes

### `GET /api/jokes`

Возвращает массив записей таблицы `jokes` в формате JSON.

Пример:

`http://127.0.0.1:8000/api/jokes`

### `GET /jokes`

Отображает страницу со списком шуток.

Возможности:

- пагинация
- сортировка по `ID`
- сортировка по `External ID`
- сортировка по `Type`
- по умолчанию сверху отображаются старые записи

Пример:

`http://127.0.0.1:8000/jokes`

## Проверка решения

1. Выполнить команду вручную:

```bash
php artisan jokes:fetch-random
```

2. Проверить JSON route:

`http://127.0.0.1:8000/api/jokes`

3. Проверить HTML-страницу:

`http://127.0.0.1:8000/jokes`

4. Проверить тесты:

```bash
php artisan test
```
